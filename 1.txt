Add Vercel Analytics and Speed Insights to this Next.js app.

Steps:

1. Install dependencies:

Run in terminal:
npm install @vercel/analytics @vercel/speed-insights

---

2. Update the root layout file:

Open:
app/layout.tsx

---

3. Add imports at the top of the file:

import { Analytics } from "@vercel/analytics/react";
import { SpeedInsights } from "@vercel/speed-insights/next";

---

4. Add components to the layout:

Inside the <body> tag, at the bottom, add:

<Analytics />
<SpeedInsights />

Example structure:

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>
        {children}
        <Analytics />
        <SpeedInsights />
      </body>
    </html>
  );
}

---

5. Deploy the app:

Run:
git add .
git commit -m "add vercel analytics and speed insights"
git push

---

6. Verify:

- Visit the deployed site
- Click around pages
- Wait ~30–60 seconds
- Check Vercel dashboard:
  - Analytics tab (for traffic)
  - Speed Insights tab (for performance)

---

Constraints:

- Do not create new pages
- Do not modify UI styling
- Only modify layout.tsx and dependencies
- Ensure imports match exact package names
